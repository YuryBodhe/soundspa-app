## Быстрые команды для проверки

```bash
# Проверка приложения
pm2 status soundspa-app

# Проверка сайта
curl -I https://soundspa.bodhemusic.com

# Проверка диска
df -h

# Проверка нагрузки
uptime

# Логи приложения
pm2 logs soundspa-app --lines 50

# Логи nginx
tail -n 50 /var/log/nginx/soundspa_error.log
```
